Important Notes for reading validation report.....

> In the report an Object will be denoted as (O), Array will be denoted as [A]
Example:

> Type mismatches - are the list of fields that might have the wrong type returned in response, as per Swagger specification.

> Missing mandatory fields - are the list of fields defined as mandatory in Swagger specification, but missing in JSON response given.

> Fields Missing in Response - are the list of elements that are defined in Swagger specification, but JSON response is missing.

> Fields Missing in Swagger- are the list of elements that are in JSON response, but missing in Swagger specification.

> Hard to notice issues - Report catches spell mistakes, misplaced object for array or misplaced array for object.
